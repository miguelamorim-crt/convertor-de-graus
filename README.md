# Conversor de Temperatura - Go

Este projeto é um aplicativo simples de linha de comando desenvolvido em Go, que converte valores de graus Celsius para Fahrenheit utilizando a seguinte fórmula:

**Fahrenheit = (Celsius * 1.8) + 32**

## Como funciona

- O usuário acessa o menu no terminal.
- Ao escolher a opção de conversão, insere o valor em graus Celsius.
- O programa mostra o resultado em Fahrenheit.

## Exemplo de uso

=== CONVERTOR DE TEMPERATURA ===
1 - converter
2 - sair
opcao escolhida: 1
quantos graus quer converter para Fahrenheit: 25.0
resultado: 77
